# Build Instructions
1. `git clone https://gitlab.com/cs354r-sasaki` 
2. `make fresh`
3. Open Godot
4. Select our project (located in `sasaki/gd-project/project.godot`)
5. Adjust any desired settings on many of our Plugin nodes, all are documented below.
